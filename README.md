# Single-Channel-Headset-Program
项目简介
本工具集是 Python LSL（Lab Streaming Layer）数据流场景的轻量辅助套件，包含「LSL 数据流查看器」与「单道连接小程序」两款工具，分别适配多维度可视化管理与单通道轻量调试的核心需求。
工具组件说明
1. LSL 数据流查看器
核心功能
通过 lsl_to_websocket_bridge.py 桥接服务，实现 Python LSL 数据流的全流程可视化管理：
自动检测并连接可用的 LSL 数据流
实时展示数据流波形、流元信息（名称 / 类型 / 通道数等）
统计数据接收样本数、传输速率等核心指标
支持滤波参数配置（如低通滤波），优化信号查看效果
功能亮点
无需手动配置流参数，自动适配可用流
可视化界面同步展示「波形 + 元信息 + 统计数据」，信息更聚合
2. 单道连接小程序
核心功能
专注单通道 LSL 流的轻量工具：
简化单通道场景下的 LSL 流连接流程（无需复杂配置）
快速读取单通道数据流，支持轻量数据预览
适用场景
适合单通道生理信号、传感器数据的快速调试（临时校验数据连通性）或轻量查看（无需全维度可视化）场景。
