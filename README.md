# Single-Channel-Headset-Program
 **LSL数据流查看器**
   - 核心功能
     通过 `lsl_to_websocket_bridge.py` 桥接服务，实现Python LSL数据流的全流程可视化
     - 自动检测并连接可用的 LSL 数据流
     - 实时展示数据流波形、流元信息（名称/类型/通道数等）
     - 统计数据接收样本数、传输速率等核心指标
     - 支持滤波参数配置（如低通滤波），优化信号查看效果
   - 功能亮点
     - 无需手动配置流参数，自动适配可用流
     - 可视化界面同步展示「波形 + 元信息 + 统计数据」，信息更聚合

2. **单道连接小程序**
   - 核心功能
     专注单通道 LSL 流的轻量工具：
     - 简化单通道场景下的 LSL 流连接流程（无需复杂配置）
     - 快速读取单通道数据流，支持轻量数据预览
